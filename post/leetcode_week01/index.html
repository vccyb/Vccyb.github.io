LeetCode 1 两数之和

要注意这个题是有个前提条件的，即每种输入只有一种答案

class Solution {
    public int[] twoSum(int[] nums, int target) {
        //1. 特判
        if(nums == null || nums.length == 0){
            return new int[]{-1, -1};
        }
        //2. 利用Map特性，存储每个点的值和所对应的下标
        Map<Integer, Integer> map = new HashMap<Integer, Integer>();
        for(int i = 0; i<nums.length; i++){
            if(map.containsKey(target - nums[i])){
                 //是否含有目标值，即target-nums[i]
                return new int[]{map.get(target - nums[i]), i}; //获取下标并返回
            }
            map.put(nums[i], i);
        }
        return new int[]{-1, -1};
    }
}

hashMap 插入，查询都是O(1),n重循环
时间复杂度O(n)

LeetCode 2 两数相加

这个题是模拟，有个小技巧就是虚拟头节点

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
class Solution {
    public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        ListNode dummy = new ListNode(0); //虚拟头节点
        ListNode cur = dummy;
        int t = 0;
        while(l1!=null || l2!= null){
            //l1 或者是 l2 没有循环完毕
            if(l1 != null){
                t += l1.val;
                l1 = l1.next;
            }

            if(l2 != null){
                t += l2.val;
                l2 = l2.next;
            }

            cur.next = new ListNode(t%10); //
            t/=10;

            cur = cur.next; 
        }
        if(t != 0) cur.next = new ListNode(t);
        return dummy.next;
    }
}

LeetCode 3 无重复字符的最长子串

本质，双指针算法，使用的前提就是单调性

class Solution {
    public int lengthOfLongestSubstring(String s) {
        Map<Character, Integer> map = new HashMap<Character, Integer>();
        int ans = 0;
        for(int i=0,j=0;i<s.length();i++){
            map.put(s.charAt(i), map.getOrDefault(s.charAt(i),0)+1);
            //判断i位置的元素是否有重复
            while(map.get(s.charAt(i))>1){
                map.put(s.charAt(j),map.get(s.charAt(j))-1);
                j++;
            }
            ans = Math.max(ans, i-j+1);
        }
        return ans;
    }
}

LeetCode 4 寻找两个正序数组中的中位数

困难题！！！
暴力基本解法
思路：开辟一个新的数组

class Solution {
    public double findMedianSortedArrays(int[] nums1, int[] nums2) {
        int[] nums;
        int m = nums1.length;
        int n = nums2.length;
        nums = new int[n+m];
        if(m==0){
            if(n%2==0){
                return (nums2[n/2 -1] + nums2[n/2]) / 2.0;
            }else{
                return nums2[n/2];
            }
        }

        if(n==0){
            if(m%2==0){
                return (nums1[m/2 -1] + nums1[m/2]) / 2.0;
            }else{
                return nums1[m/2];
            }
        }

        int count = 0;
        int i=0,j=0;
        while(count!=(m+n)){
            if(i==m){
                while(j!=n){
                    nums[count++] = nums2[j++];
                }
                break;
            }

            if(j==n){
                while(i!=m){
                    nums[count++] = nums1[i++];
                }
                break;
            }

            if(nums1[i]<nums2[j]){
                nums[count++] = nums1[i++];
            }else{
                nums[count++] = nums2[j++];
            }
        }

        if(count%2==0){
            return (nums[count/2 - 1]+ nums[count/2]) / 2.0;
        }else{
            return nums[count/2];
        }

    }
}

时间复杂度O(m+n) 还开辟了新的数组，不符合题目要求

递归做法
思路转化为找第k小的数，每次可以省略一般2/k的数

class Solution {
    public double findMedianSortedArrays(int[] nums1, int[] nums2) {
        int total = nums1.length + nums2.length;
        if(total % 2 == 0){
            int left  = f(nums1, 0, nums2, 0, total/2);
            int right = f(nums1, 0, nums2, 0, total/2 + 1);
            return (left+right) / 2.0;
        }else{
            return f(nums1, 0, nums2, 0, total/2+1);
        }
    }


    static int f(int[] nums1, int i, int[] nums2, int j, int k){
        if(nums1.length - i > nums2.length - j) return f(nums2,j,nums1,i,k);

        if(nums1.length==i) return nums2[j+k-1]; //序列1用完了，直接返回
        if(k==1) return Math.min(nums1[i],nums2[j]); //最终k==1 了 比较一下返回
        int si = Math.min(nums1.length, i + k/2), sj = j + k- k/2;
        if(nums1[si-1] > nums2[sj-1]){
            return f(nums1,i,nums2,sj,k-(sj-j));
        }else{
            return f(nums1,si,nums2,j,k-(si-i));
        }
    }
}

LeetCode 5 最长回文子串

思路：1、枚举数组中的每个位置i，从当前位置开始向两边扩散，
2、当回文子串是奇数时，从i开始往两边扩散，
3、当回文子串是偶数时，从i，i + 1开始往两边扩散
4、找到以i为中心的最长回文子串的长度，若存在回文子串比以前的长，则更新回文串的长度maxv，以及两端的位置l和r

class Solution {
    public String longestPalindrome(String s) {
        if(s==null||"".equals(s)) return s;
        String res = Character.toString(s.charAt(0));
        for(int i=0;i<s.length();i++){
            //回文长是奇数
            int left = i-1, right = i+1;
            while(left>=0 && right < s.length() && s.charAt(left)==s.charAt(right)){
                if(right - left + 1> res.length()){
                    res = s.substring(left, right+1);
                }
                left--;right++;
            }

            //偶数情况下
            left = i;right = i+1;
            while(left>=0 && right < s.length() && s.charAt(left)==s.charAt(right)){
                if(right - left + 1> res.length()){
                    res = s.substring(left, right+1);
                }
                left--;right++;
            }
        }
        return res;
    }
}
